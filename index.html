📐 รูปแบบสมการมาตรฐาน

y = A sin(Bx - C) + D

📏 แอมพลิจูด (A)

A = แอมพลิจูด1.00
A ควบคุมความสูงของกราฟ
A > 1 → ยืดแนวตั้ง   A < 1 → หดแนวตั้ง
ค่าสูงสุด = A, ค่าต่ำสุด = −A

🔄 ความถี่ / คาบ (B)

B = ความถี่เชิงมุม1.00
B ควบคุมความถี่ (T = 2π/B)
B > 1 → หดแนวนอน (คาบสั้น)
B < 1 → ยืดแนวนอน (คาบยาว)

↔️ การเลื่อนเฟส (C)

C = เฟส (π)0.00π
C > 0 → เลื่อนขวา (เพราะสูตรคือ Bx - C)
C < 0 → เลื่อนซ้าย
การเลื่อน = C/B หน่วย

⬆️ เลื่อนแนวตั้ง (D)

D = ขึ้น/ลง0.00
D > 0 → เลื่อนขึ้น   D < 0 → เลื่อนลง
Midline อยู่ที่ y = D
